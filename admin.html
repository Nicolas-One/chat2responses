Chat2Responses

Chat Completions → Responses API 转换代理

⚙ 上游与模型

所有请求将被转发到此上游地址的 /chat/completions
配置后将优先使用此 Key，否则透传请求中的 Authorization Header
匹配的模型会做 reasoning 格式转换；不匹配则直接透传
开启后，所有请求的 reasoning 内容将被移除，即使客户端要求 reasoning 也无效
开启后日志写入 chat2responses.log；关闭后仅调试模式输出到控制台

↪ 模型别名 (Model Alias)

别名 (请求名) 实际模型 (上游名)

↬ 推理力度映射 (Reasoning Effort)

请求值 (如 low) 映射值 (如 high)
